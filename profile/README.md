# Nevada Supreme Court Administrative Office of the Courts (AOC)

## About

The Administrative Office of the Courts (AOC) provides administrative support and services to the Nevada Supreme Court, Court of Appeals, and the state's district courts. This repository contains software projects, tools, and resources developed and maintained by the AOC to support the Nevada judiciary system.

## Mission

To improve the administration of justice in Nevada through innovative technology solutions, efficient case management systems, and reliable court services.

## Repository Overview

This repository hosts various applications and tools that support:

- **Court Operations**: Case management and administrative systems
- **Public Services**: Tools and services for court users and the public
- **Data Management**: Systems for maintaining and analyzing court data
- **Reporting**: Tools for generating court statistics and reports
- **Automation**: Scripts and workflows to improve operational efficiency
